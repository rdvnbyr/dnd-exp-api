Node-Express Runtime Application

This is a Node-Express Runtime Application. It is a simple application that demonstrates the use of Node-Express to create a REST API and a GraphQL API.

Rest Explorer

The Rest Explorer allows you to explore the REST API. You can use the Rest Explorer to create, read, update, and delete.

GraphQL

The GraphQL API allows you to create, read, update, and delete.

Feedback

The Feedback page allows you to submit feedback. The feedback is stored in a MongoDB database.
